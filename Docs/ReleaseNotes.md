# Razix Engine Release Notes

#### Version - 0.50.0 [Development] - 26/11/2025
- Re-wrote the RHI backend in C99 with full VK and DX12 backend support, can easily switch backend using config files
- loads of custom STL (RZDynamicArray, RZHashMap etc.)
- removed redundant stuff and improved rendering
- Added lot of engine/gfx tests
- loads of stablitiy and c improvements
- Added design and moodboard for Tanu game, will be focusing more on it
- Improved stability for engine with tests and full linux support
- Added full dualsense support
- added custom std and stl headers (hashmap, sprintf, type_traits etc.)
- Auto resource views management by frame graph and shader bindmap for descriptor tables
- Improved resource system design for RHI and GFX backend
- simplified code and removed unwanted stuff, removed QTEditor and redundant dependencies and configs
- Added config files for engine/renderer and memory systems
- Added mood board and roadmap updates for the game "Tanu"
